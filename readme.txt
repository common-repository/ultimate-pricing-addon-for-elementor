== Description ==

This is a simple price addon for elementor. After install this plugin you will get a WPP price widget in the elementor widget area. You can create your own custom price table. All of these options are highly customizable.

== Feature ==
* Fully responsive
* Easy to customizable
* Possible to customize own color, fonts etc
